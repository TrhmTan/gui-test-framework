navigate: navigate to URL (Target: URL key trong PAGE, Value/Expected: URL fallback)
click: Click vào phần tử UI (Target: ID của phần tử trong ELEMENT)
input: Nhập dữ liệu (Target: ID phần tử trong ELEMENT, Value: Giá trị nhập hoặc tham chiếu dữ liệu)
clear: Xóa dữ liệu trong trường input (Target: ID phần tử trong ELEMENT)
select: Chọn một tùy chọn trong dropdown (Target: ID phần tử trong ELEMENT, Value: Giá trị tùy chọn)
check_status: Xác thực trạng thái/văn bản hiển thị của phần tử UI (Target: ID phần tử, Expected: visible, hidden, enabled, disabled, checked, unchecked, hoặc nội dung văn bản cần kiểm tra)
upload_file: Upload file qua phần tử input type=file (Target: ID phần tử, Value: Tên tệp tin trong data/assets)
scroll_to: Cuộn trang đến khi hiển thị phần tử UI (Target: ID phần tử)
hover: Rê chuột lên phần tử UI (Target: ID phần tử)
press_key: Nhấn một phím nóng từ bàn phím (Target: ID phần tử, Value: Tên phím như Enter, Tab, Escape)
capture: Chụp ảnh màn hình bước test hiện tại (Value: Thông tin mô tả ảnh chụp)
check_value: Xác thực nội dung thực tế của phần tử UI bằng ngôn ngữ tự nhiên (hỗ trợ textbox txt_/searchbox, dropdown ddl_/select_, checkbox chk_, radio rad_, label lbl_, error_msg, toast...) so với giá trị kỳ vọng.
wait_for: Chờ element/page đạt trạng thái chỉ định — THAY THẾ waitForTimeout (Target: element_id hoặc "page", Value: visible|hidden|attached|detached|networkidle|domloaded|load)
assert_url: Kiểm tra URL hiện tại khớp with pattern mong đợi — hỗ trợ wildcard (*), contains, hoặc exact match (Expected: URL pattern)
dblclick: Double click vào phần tử UI (Target: ID của phần tử trong ELEMENT)
set_context: Lưu giá trị vào Shared Runtime Context (Target: tên key, Value: giá trị cần lưu hoặc tham chiếu $data_)
get_context: Đọc giá trị từ Shared Runtime Context và fill vào element (Target: tên key, Value: element_id để fill — bỏ trống nếu chỉ đọc)
capture_text: Chụp text từ UI element và lưu vào Shared Runtime Context (Target: element_id, Value: tên key để lưu)
